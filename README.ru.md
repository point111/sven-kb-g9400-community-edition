# SVEN KB-G9400 Community Patch

Неофициальный патч совместимости для официального конфигуратора клавиатуры SVEN KB-G9400 в Windows 11.

## Назначение версии 1.x

Версия 1.x только восстанавливает работу фирменного программного обеспечения в Windows 11. Она не добавляет новых функций клавиатуры и не включает русификацию интерфейса.

## Проверено

- Windows 11 25H2;
- устройство `VID_30FA&PID_2350`;
- запуск конфигуратора без ошибки `Product and driver are inconsistent`;
- изменение подсветки;
- переназначение клавиш;
- сохранение INI-профилей;
- повторная установка;
- полный откат к оригинальному EXE;
- автоматическая проверка целостности.

## Требования

Сначала установите официальный конфигуратор SVEN KB-G9400 в стандартный каталог:

```text
C:\Program Files (x86)\KB-G9400 Gaming Keyboard
```

Репозиторий не содержит оригинальный EXE, MSI или фирменные ресурсы SVEN.

## Установка

Откройте PowerShell от имени администратора в каталоге репозитория:

```powershell
Set-ExecutionPolicy -Scope Process Bypass
.\build\install.ps1
.\build\test.ps1
```

## Восстановление

```powershell
.\build\restore.ps1
.\build\test.ps1
```

## Дальнейшее развитие

Новые возможности, отсутствующие в фирменном ПО, планируются для версии 2.x. Локализация и другие изменения интерфейса могут потребовать контролируемого патчинга ресурсов оригинального EXE либо отдельного загрузчика/компонента, который проверяет окружение и применяет только совместимые расширения.

## Правовой статус

Это независимый community-проект, не связанный и не одобренный SVEN. Названия и товарные знаки принадлежат их владельцам.
