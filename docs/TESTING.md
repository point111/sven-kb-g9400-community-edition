# Проверка релиза

- Установка на чистую Windows 11 с официальным пакетом.
- Проверка резервной копии.
- Проверка повторной установки.
- Переключение RU/EN с перезапуском.
- Подсветка: цвет, яркость, скорость, режим.
- Переназначение одной клавиши.
- Макрос на тестовой клавише.
- Переподключение USB.
- Восстановление оригинала.
- Удаление патча.
