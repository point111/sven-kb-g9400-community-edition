# Техническое описание

Официальный конфигуратор распознаёт `VID_30FA&PID_2350`, открывает vendor-specific HID collections и выполняет дополнительную проверку идентификационных данных устройства. На исследованной клавиатуре HID-протокол полностью совместим, но идентификационная строка не совпадает с ожидаемой, из-за чего программа прекращает запуск.

Патч меняет один условный переход в установленном EXE:

- offset: `0x669B`
- original: `0x74`
- patched: `0xEB`

Изменение отключает только отказ после проверки идентичности. HID-команды, настройки подсветки и запись профилей не изменяются.

Публичный проект не должен включать оригинальный исполняемый файл или полный набор ресурсов SVEN.
