# План проверки подсветки по языку ввода

## Фаза 1 — наблюдение

- определить события Windows, связанные со сменой раскладки;
- проверить переключение RU/EN в браузере, блокноте, терминале и приложениях с отдельной раскладкой;
- убедиться, что обработчик не создаёт постоянную высокую нагрузку.

## Фаза 2 — HID-прототип

- записать одну безопасную смену статического цвета штатным конфигуратором;
- выделить минимальную последовательность HID-команд;
- воспроизвести только её на тестовой клавиатуре;
- проверить восстановление после отключения USB.

## Фаза 3 — интеграция

- связать RU и EN с двумя пользовательскими цветами;
- добавить задержку и подавление повторных команд;
- запретить работу при открытом штатном конфигураторе;
- добавить выключатель автозапуска.

## Условия остановки

Исследование прекращается, если команды требуют полной перезаписи профиля, вызывают зависание устройства или конфликтуют со штатным ПО.
