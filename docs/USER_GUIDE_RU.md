# Руководство пользователя

## Назначение

Community Patch исправляет ошибочный отказ `Product and driver are inconsistent` в официальном конфигураторе SVEN KB-G9400. Проект не заменяет драйвер и не распространяет фирменные файлы SVEN.

## Подтверждённая конфигурация

- Windows 11 25H2;
- клавиатура `VID_30FA&PID_2350`;
- официальный конфигуратор установлен в `C:\Program Files (x86)\KB-G9400 Gaming Keyboard`.

Другие версии и аппаратные ревизии считаются непроверенными.

## Установка патча

1. Закройте официальный конфигуратор.
2. Откройте PowerShell от имени администратора.
3. Перейдите в каталог репозитория.
4. Выполните:

```powershell
Set-ExecutionPolicy -Scope Process Bypass
.\scripts\Install-CommunityPatch.ps1
```

Скрипт проверяет байт в известной позиции, создаёт резервную копию, сверяет её SHA-256, атомарно записывает изменение и повторно проверяет результат.

## Повторный запуск

Повторный запуск установщика безопасен: если исправление уже применено, файл повторно не изменяется.

## Восстановление оригинала

```powershell
.\scripts\Restore-Original.ps1
```

Перед завершением скрипт проверяет резервную копию и SHA-256 восстановленного файла.

## Диагностика

Журнал создаётся рядом с резервной копией:

```text
C:\Program Files (x86)\KB-G9400 Gaming Keyboard\.community-patch-backup\community-patch.log
```

Файл `state.json` содержит контрольные суммы оригинального и текущего EXE.

## Русская локализация

Проверенный словарь находится в `localization/ru-RU.csv`. Он является источником терминологии для текстовых и растровых ресурсов. Готовый RU-payload пока не публикуется: формат ресурсов должен быть воспроизводимо извлечён и собран без включения оригинальных файлов SVEN.

## Типичные ошибки

### Официальный конфигуратор не найден

Проверьте, что установлен официальный пакет и каталог имеет ожидаемое имя.

### Неизвестная версия EXE

Скрипт остановился до изменения файла. Не пытайтесь менять offset вручную; приложите SHA-256 EXE, версию Windows и Hardware IDs клавиатуры к issue.

### Резервная копия не прошла проверку

Не удаляйте и не заменяйте файлы внутри `.community-patch-backup`. Сохраните журнал и сообщите об ошибке.

## Данные для отчёта об ошибке

- версия Windows (`winver`);
- Hardware IDs клавиатуры;
- SHA-256 официального EXE;
- содержимое `community-patch.log`;
- точный текст ошибки.

Не прикладывайте оригинальный EXE, MSI или архив производителя к публичному issue.
