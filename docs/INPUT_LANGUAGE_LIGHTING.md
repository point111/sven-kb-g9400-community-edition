# Подсветка по языку ввода

Идея: менять цвет или профиль подсветки клавиатуры при переключении активной раскладки Windows.

## Предполагаемая архитектура

Небольшое фоновое приложение:

1. определяет активное окно;
2. получает язык ввода его потока;
3. реагирует только на фактическое изменение раскладки;
4. отправляет клавиатуре заранее проверенную HID-команду;
5. не запускает штатный конфигуратор и не изменяет прошивку.

## Риски

- язык ввода относится к активному потоку или окну, а не является единым глобальным значением;
- разные приложения могут переключать раскладку независимо;
- частые HID-команды могут создавать задержки или конфликтовать с конфигуратором;
- пока неизвестен минимальный пакет, который меняет только текущий цвет без перезаписи остальных настроек.

## Статус

Research / эксперимент. Функция не входит в обязательный состав v1.0.0 и будет реализована только после безопасного прототипа.
